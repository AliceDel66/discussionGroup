## 2. 项目背景摘要

### 产品定位

曜承 HR Agent 是 **Agent Native 招聘工作流产品**：以招聘邮箱 / 手动上传 / BOSS 邮件投递为简历入口，自动完成解析、JD 匹配、风险识别、面试问题生成，输出标准评估卡，**AI 只做辅助，人工做最终决策**。

核心原则（全版本适用）：

> AI 只做招聘辅助与信息整理，不自动淘汰、不自动录用、不绕过候选人授权、不替代 HR / CEO / 面试官做最终判断。
